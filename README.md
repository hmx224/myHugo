# myHugo
hugo搭建blog

## 安装皮肤

```cgo

cd themes
git clone https://github.com/flysnow-org/maupassant-hugo

```
## 运行Hugo

```cgo

hugo server --theme=maupassant-hugo --buildDrafts

```
